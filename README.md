## Note: 
- For build, tested only with GCC of Mingw-w64 v12.1.0
- For use, tested only on Windows 11 with Windows Terminal v1.15.2875.0
- Questions? Read the source, for it's short

## Build
```PowerShell
gcc main.c -o sii
```

## Example
```PowerShell
# PowerShell
sii `
input.jpg <# Input Image path #> `
50        <# Output width #>
```
